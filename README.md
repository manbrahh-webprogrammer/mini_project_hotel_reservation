# Hotel Reservation System

A web-based hotel booking platform using PHP and MySQL.

## Features
- User registration & login via email
- Room reservation with QR payment
- Admin dashboard to approve/reject bookings
- Automatic room availability updates

## Technologies
- PHP
- MySQL (phpMyAdmin)
- HTML, CSS, JavaScript
